# Architectural Regression Checklist

- [ ] No background tabs are created.
- [ ] No second Discord authentication is required.
- [ ] Passive discovery remains functional.
- [ ] Export remains human- and AI-friendly.
- [ ] Discord DOM selectors remain inside the Discord adapter.
- [ ] Core modules do not query Discord DOM structures.
- [ ] Navigation, collection, parsing, and export remain separable.
- [ ] Identity provenance is preserved.
- [ ] Released milestone package includes source, documentation, requirements, changelog, handover, and validation results.
